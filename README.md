# Task Tracker
> This is a project from the YouTube crash course
## Project setup
```
npm install
```
## Run the JSON-Server backend(on port 5000)
```
npm run backend
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
